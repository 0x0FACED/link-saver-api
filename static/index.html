Save your links with Linkerify!

Send the bot the links you want to save! You will be able to retrieve the content of these links even if the original link becomes unavailable.

Bot Commands:

  • /start - show start menu
  • /save - show help on saving
  • /get - show help on getting data from the bot
  • /help - general help
  • /list - get a list of links for further content retrieval
  • /del - get a list of links for further deletion from the server
